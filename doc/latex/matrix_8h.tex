\subsection{Detailed Description}
Function declarations for matrix operations. 

Matrices are stored in 1-\/D arrays. To access a cell, use \mbox{[}r$\ast$n+c\mbox{]} instead of \mbox{[}r\mbox{]}\mbox{[}c\mbox{]}.
